# zstd scaffold

Scaffold repository for https://github.com/facebook/zstd.git.

Contains files to work with workspace scaffolding system.

- `DEPS` points to https://github.com/facebook/zstd.git
- `scaffold.lua` points to `zstd.lua` and returns the scaffold package.
- `zstd.lua` defines the scaffolded projects and variables to export.
